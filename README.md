# A Tool for Muting Bluesky Content & Tags

[This tool](https://colarusso.github.io/bluesky-mute/) let's you quickly mute a word or phrase in Bluesky. It comes with a bookmarklet you can use to do this from any page. You'll have to add an app password and info for your account (handle and PDS server) before using the bookmarkelt, but once you do, just select text and click the bookmarklet. Note: you can can change the default mute time and subsequent actions will remember your last selection.
